Pokemon Card Image Detector

Project Introduction

Welcome to our Pokémon card detection demo. This project is specifically designed to recognize Pokémon cards from the Base Set. Although it is trained primarily on French Base Set 2 cards, it can effectively detect all Base Set cards, even those in English.

Utilizing advanced machine learning algorithms, our tool extracts essential details from each card, including name, number, set, and year. We hope this demo showcases the potential of image recognition technologies in the world of collectible card games.
